Bash scripting - Cheat sheet

Here you will find an introductory table presenting you with the basic commands required to use a shell terminal in the Bash scripting language

What is Bash scripting?

Bash scripting is using the Bash shell to automate tasks on Unix-like systems by writing commands in a script file.
It supports variables, loops, and conditionals, making it powerful for system administration, automation, and file processing.

Name Command Action Option
List ls The ls command is used to list the contents of a directory.
It displays directories, files and their information		 -l Long listing
-a Include hidden files
-t Sort by modification time
-h Human-readable sizes
-R List subdirectories recursively
-1 List one file per line
-d List directories themselves, not their content
Change Directory cd The cd command is used to change the current working directory in the terminal .. Move up one directory level
~ Change to the home directory
- Switch to the previous directory
/ Change the root directory
Print Directory pwd The pwd command is used to show you the full path of the directory you're currently in -L Display the logical current working directory
-P Display the physical current working directory structure
Echo echo The echo command is used to show a line of text or a variable in the terminal -n Don't add a new line at the end of the output
-e Allow special characters like i.e. '\n' for new lines
-E Don't allow special characters (default)
Concatenate & Display cat The cat command is used to show the content of files in the terminal and to combine multiple files into one -n Add numbers to each line
-b Add numbers only to lines with text
-s Remove extra empty lines from output
-v Show non-printing characters (hidden characters)
Copy cp The cp command is used to copy files and directories from one location to another -r Copy all files and folders inside a directory
-i Ask before replacing files
-u Copy only if the source is newer than the destination file
-v Verbose mode: show files being copied
Move or Rename mv The mv command is used to move or rename files and directories -i Ask before replacing files
-u Move only if the source is newer than the destination file
-v
Verbose mode: show files being moved
Remove rm The rm command is used to delete files or directories -r Delete a folder and everything inside it
-i Ask before deleting each file
-f Force delete without asking
-v Verbose mode: show files being deleted
